# Employee Info Fetcher

An intelligent employee information retrieval system powered by CrewAI. This project uses multi-agent AI to query and analyze employee data from a knowledge base, providing detailed summaries and insights about team members. Simply input an employee name and the system leverages AI agents to retrieve comprehensive information including background, experience, and professional details.

## Setup

Install dependencies with `pip install -r requirements.txt`, add your OpenAI API key to a `.env` file, then run `python crewai_app.py` and enter an employee name to fetch their information.
